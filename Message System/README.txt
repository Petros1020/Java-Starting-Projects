The application is an message system between the members of the Peaky Blinders
family!!!

You can sign in with severl users, each one with different privilages:

Username - Password 
admin – admin (administrator)

You can send an message to another member of the family and according to 
which privilages you are granted, you can read/edit/delete all messages,
or even create and delete users!
