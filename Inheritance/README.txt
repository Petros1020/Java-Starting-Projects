Project Humanity
We will model in our first version Students and Workers.
The software calculates the money a worker earns by hour.

The First Name starts with a capital letter and it is more than 2 symbols.
The Last Name starts with a capital letter and it is more than 3 symbols.
If first letter is not a capital, the software fixes it.
Faculty Number is in range of [5...10] symbols
Week Salary is more than 10
Working Hours are in range of [1...12]
Two digits exactly are printed after every double value's decimal separator.
The Faculty Number should is consisted only of digits and letters.
