A web application using Tomcat and Servlets.

You can upload songs and the application is searching for title,
artist, album, lyrics, cover photo etc. If it does not find anything,
you can also update them.

A table with all records appears, after the input. You can delete songs
or edit them there.

The inputs are stored in MySQL database as Blobs and Json library
is used to handle the lyrics to display the lyrics.
