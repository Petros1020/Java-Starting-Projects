This is a web application made with Spring Boot. It is an application like
meetup.com. 
You can create an account and search for activities to join!
You can create an activity, with description, date, time, location and meet
with people which you share the same interests and ideas to have fun, learn
and socialize!

This application was made in java, uses an online server to deliver the 
uploaded photos and the technologies used were Spring Boot, Hibernate, MYsql,
JPA, CRUDrepository, JSP, JSTL, Bootstrap, HTML, CSS, AJAX and WebSockets.
